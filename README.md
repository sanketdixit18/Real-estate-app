# 🏡 Real Estate Price Prediction & Recommendation System

A Machine Learning web application that predicts property prices and recommends similar properties based on location, amenities, and property features.

## 🚀 Features

- Property Price Prediction
- Similar Property Recommendation
- Interactive Streamlit Web App
- Data Cleaning & Feature Engineering
- Exploratory Data Analysis (EDA)
- Machine Learning Model Training
- Model Evaluation
- Hybrid Recommendation System

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sanketdixit18/Real-estate-app.git
```

Go into the project

```bash
cd Real-estate-app
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the app

```bash
streamlit run app.py
```
